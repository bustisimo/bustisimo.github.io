# bustisimo.github.io
This is my website.

It's written in HTML and CSS right now.

The "Click Me" button on the home page uses the free [CounterAPI](https://counterapi.dev) service to keep a global click count. The JavaScript fetches `https://api.counterapi.dev/get/<namespace>/<key>` to read the count and `https://api.counterapi.dev/hit/<namespace>/<key>` to increment it.
